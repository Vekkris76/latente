# LATENTE — Objetivo

Construir un sistema de co-presencias latentes (anti-engagement) que:
- NO usa ubicación exacta ni guarda coordenadas.
- Funciona por patrones (no por eventos sueltos).
- Propone ventanas de sincronía.
- Solo revela identidad si hay reconocimiento mutuo.
- Evita feeds, swipes, rankings y boosts.

Este repositorio empezará con especificación funcional y estado del sistema.
No se implementa código hasta que la spec esté cerrada.
