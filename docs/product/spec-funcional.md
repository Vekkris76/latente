# LATENTE — Especificación funcional (v1)

## Flujo base
1) Usuario activa observación pasiva (consentimiento).
2) Sistema detecta co-presencias por patrones.
3) Se propone una ventana futura (aceptar / declinar).
4) Durante ventana activa existe un único botón: “Creo que te he visto”.
5) Revelación solo si ambos confirman.
6) Si no es mutuo: no hay rechazo explícito, se purga.

## Prohibiciones
- No mapas
- No perfiles visibles iniciales
- No feed
- No ranking
- No pagos por prioridad

## Pantallas mínimas
- Onboarding (qué es)
- Consentimientos
- Perfil mínimo
- Home (estado + ventanas)
- Ventana activa
- Revelación
- Seguridad (bloquear/reportar)
- Eliminar cuenta
