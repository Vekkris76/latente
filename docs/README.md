# LATENTE (MVP) — CLI v1

CLI funcional para validar el flujo core sin HTTP, sin DB y sin cron.
Principios: anti-GPS, anti-engagement, sin feed, sin ranking, sin boosts.

## Requisitos
- Node.js + npm

## Instalar
npm install

## Tests
npm run test:unit
npm run test:integration

## Ejecutar CLI
npm run cli
